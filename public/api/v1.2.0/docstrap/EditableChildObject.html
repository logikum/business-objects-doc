Class: EditableChildObject

EditableChildObject

Represents the definition of an asynchronous editable child object.

new EditableChildObject(parent, eventHandlers)

Creates a new asynchronous editable child object instance.

The name of the model type available as: <instance>.constructor.modelType, returns 'EditableChildObject'.

Valid parent model types are:

 * EditableChildCollection
 * EditableRootObject
 * EditableChildObject
Parameters:
Name Type Argument Description
parent object

The parent business object.
eventHandlers bo.shared.EventHandlerList <optional>

The event handlers of the instance.
Fires:
  • EditableChildObject#event:preCreate

  • EditableChildObject#event:preFetch

  • EditableChildObject#event:preInsert

  • EditableChildObject#event:preUpdate
  • EditableChildObject#event:postUpdate
  • EditableChildObject#event:preRemove
  • EditableChildObject#event:postRemove
Throws:
  • Argument error: The parent object must be an EditableChildCollection, EditableRootObject or EditableChildObject instance.
  • Argument error: The event handlers must be an EventHandlerList object or null.
